# Submit Review

Submit your review and pass the turn to Ralph.

## Command

```bash
./mini-skill/io.sh submit-lisa "[TAG] summary

detailed content..."
```

## Required Format

Your content MUST start with a tag and one-line summary:

```
[TAG] One line summary

Detailed content here...
```

## Valid Tags for Lisa

| Tag | When to Use |
|-----|-------------|
| `[PASS]` | Work approved, can proceed |
| `[NEEDS_WORK]` | Issues found, needs changes |
| `[DISCUSS]` | Responding to Ralph's argument |
| `[QUESTION]` | Asking for clarification |
| `[CONSENSUS]` | Confirming agreement |

## After Submission

The turn automatically passes to Ralph. You must STOP and wait.
